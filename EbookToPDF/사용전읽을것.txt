혹시 몰라서 추가로 적어놓을게
이런식으로 개인이 ebook을 이미지, PDF로 만들고 공부용으로 쓰는것까지는 합법인데 만든 이미지, PDF를 공유하면 불법
현재 프로젝트 명은 EbookToPDF인데 아직 귀찮아서 이미지까지만 해놨음

EbookToPDF(아님)사용법
	일단 ebook앱에서 페이지 넘기는 기능이 방향키 오른쪽이어야 함
	교보 기준으로 만든거긴 한데 다른키가 필요한 경우 소스째로 넘기니까 수정해서 쓰면 될듯

SelectedMonitor 콤보박스
	이거 다중모니터 기준으로 캡쳐딸 때 모니터가 뭔지 알아야 될 거 같아서 따놨는데 신경안써도 될듯

영역지정
	영역지정 버튼 누르면 Dlg가 반투명으로 모든 화면 (트리플모니터 기준으로 모니터 3개 가로세로 합산)사이즈로 확장될건데
	그냥 필요한 위치의 StartPos에서 마우스 Down / EndPos에서 마우스 Up
	이러면 StartPos / EndPos에 좌표가 채워짐

캡쳐
	1회성 캡쳐로 MakeDirName에 적혀있는 폴더가 프로젝트 폴더 내에 생성이 되고, Image0000.bmp형식으로 생성됨

CaptureDelay
	초단위로 입력 페이지 넘기기 -> 캡쳐 두개의 사이에서 대기하는 시간

MaxCount
	캡쳐를 몇번 할 것인지 설정
